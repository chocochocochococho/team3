## 주요 사용 터미널 명령

- `npm start` : 개발서버 진입
- `ctrl + c` : 터미널 종료
- `git pull` : pull(작업 전, 커밋 전 필수)
- `npm i` : 작업 전, 패키지 설치 업데이트(없을 수도 있음)
- 작업 전 `git pull` -> `npm i` 진행 후 작업 부탁드림니다 아니면 충돌날 수 있어요!! 그러면.. 작업한게 모두 날라갈 수 있는 매쥑..
